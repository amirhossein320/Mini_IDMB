# Mini_IDMB

### An Android project using OMDB API, the project can store data and use it in offline mode

# Implemented by MVI Architecture
- Kotlin
- Manual Dependency Injection
- Manual Navigation Handling
- ViewModel
- Flow
- Channel
- Coroutine
- Retrofit
- Room
- Glide
